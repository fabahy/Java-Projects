Cách biên dịch chương trình: (Lưu ý đã cài đặt Java)
+ B1: Mở cmd hoặc PowerShell bên windown hoặc terminal của Linux/MacOS tại đường dẫn 20120298/Source/src
+ B2: Compile file java thành file class sử dụng commandline: javac -encoding UTF8 -cp ..\..\Libs\* *.java
+ B3: Tạo file jar từ các file class sử dụng commandline: jar cfm <FileName>.jar .\META-INF\MANIFEST.MF *.class
+ B4: Mở file jar để chạy chương trình bằng commandline: java -jar .\<FileName>.jar
